# RD_Fuzzer

사용 언어
- C++

환경
- Windows

기능
- File Format Fuzzer
- RDP Fuzzer based on rdesktop source code
- Packet Sniffing
- Network Fuzzer using Packets.

Relese
- Tester : exe 파일
- 그 외 : dll 파일

how to use
1. Modify .yaml file(RD_File_Fuzer or RD_Net_Fuzzer or RD_RDP_Fuzzer or RD_Sniffer)
2. Execute RD_Tester.exe
3. Read Usage
4. Use
ex) CMD> RD_Tester.exe 1
